# Airbnb Listings Project

This project displays Airbnb listings using HTML, CSS, and JavaScript.

The page loads Airbnb listing data from a JSON file using fetch() and async/await.

## Features

- Displays first 50 Airbnb listings
- Shows listing images
- Shows host information
- Shows amenities
- Live search feature
- Save/Favorite button

## Technologies Used

- HTML
- CSS
- JavaScript
- JSON
- Fetch API
